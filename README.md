# odin-knights-travails
Breadth-First Search on a Graph

Submission for Knight Travails project from [The Odin Project](https://www.theodinproject.com/lessons/ruby-knights-travails).

Uses a Breadth-First Search to find the shortest paths a knight on a chess board can take to any given point on the board.  

Knight class methods:  

#possible_moves - Returns all possible next moves for the given position.  
#knight_moves - Returns the shortest path from a given position to another given postion.
